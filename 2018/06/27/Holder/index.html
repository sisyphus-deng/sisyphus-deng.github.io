Installation

Open Automator and select New Document at the left lower corner of the pop window. Then choose Service.

Service receives selected –> image file in –> Finder

At left side in Actions tab, search Run Shell Script and drag it to right. On the upper left corner of the action, Shell –> /bin/python. On the upper right corner of the action, Pass input –> as arguments. Then copy and paste codes below.

Remember to use your the Dropbox token, which can be applied from Dropbox’s developer page.
