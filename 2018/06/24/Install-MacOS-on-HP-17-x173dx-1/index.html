Install MacOS/Hackintosh on a HP 17-x173dx laptop (1)

Jun 24, 2018 | Install and Maintain Hackintosh

Preface

When I come to US 10 years ago, my PhD supervisor was a big fan of Apple. All the computers in the lab is Mac. Since then, I was used to MacOS and already setup a productive pipeline for my daily use. Also, I have paid some on softwares, such as Alfred, omniGraffle, omniFocus, etc. I think windows 10 is also great OS, however, for me it was too time-consuming to switch back to windows.

Recently, I feel 8gb+256gb of my MacBook Pro MF840 (Retina, early 2015) is not enough for me and MacBook Pro is very hard and expensive to update. I planned to by a Mac mini. While I am waiting for a deal, I found this HP 17-x173dx in my storage room. I bought it last year at best buy, just because it was cheap, but I have never used it. After I briefly checked the specs, it might be great for Hackintosh.

Actually, I am not familiar with Hackintosh, and my purpose is easy – build up a MacOS workstation at home. Thus, I didn’t make everything working today.

Not working

  • Wireless
    • Realtek RTK8188EE
    • There is no support for Realtek yet
    • I used is as a desktop so that it is not a big deal. Maybe fix it with BT4.0 later.
  • Camera
  • Audio
    • Partially works
    • The max volume is a little low
    • It does not auto switch to headphone when plugged
    • The voodooDdaSettingsLoader has some permission problems

Specs

  • i7-7500U
  • 8 GB DDR4
  • Intel HD Graphics 620
  • Samsung EVO 850 1 TB
    • It comes a 1 TB Blue disk 5400rpm.
    • I am planing buy a hard drive caddy and put the old hard drive for larger
  • 17.3’’ HD+ LED (1600 x 900)

We need determine some details of the computer. I used a USB drive with live linux to do this (Maybe it can be done in windows but I am not familiar with windows and I happen to have a Manjaro as live linux drive burned by etcher . Then start linux on the computer. Open a terminal and input commands below.

Check information about audio, wireless, graphics…

Check keyboard and trackpad information

We found more details, so that we can download kext (driver) later.

  • Ethernet RLT8101/2/6E, there is a drive.
  • Wireless RLT8188EE
  • Sound card was Intel Sunrise Point-LP HD Audio
  • PS/2 trackpad and keyboard

Process

Basically, follow the installation guide on tonymax86. We need a USB drive (>16gb) and a set of USB keyboard/mouse, before we install the PS/2 drive we can’t use the keyboard and trackpad on the laptop.

  1. Download High Sierra from App Store

  2. Format the USB driver
    Using command line can prevent some GUID problems, especially the usb drive was formatted and used in windows or other system before.

  3. Download UniBeast and run it, I got UniBeast 8.3.2 on 06/24/18. UniBeast will created a installation disk for you.
    3.0 You need a tonymacx account to download it.
    3.1 I selected UEFI Boot Mode.
    3.2 Don’t select anything Graphics Configuration.
    3.3 Maybe take 10-20mins to get it done.

  4. Copy MultiBeast, KextBeast, EFI Mounter, RealtekRTL8100 driver, Voodoo PS/2 controller driver to the USB drive.
    4.1 MultiBeast will install the bootload and drivers for you after your Hackintosh installed. I got MultiBeast 8.3.2 on 06/24/18.
    4.2 KextBeast is a kext installer. I got KextBeast on 06/24/18.
    4.3 EFI Mounter will mount EFI drive and we can edit the clover config. I got EFI Mounter 3 on 06/24/18.
    4.4 I got RealtekRTL8100 driver 2.0.0 on 06/24/18.
    4.5 I got Voodoo PS/2 controller driver on 06/24/18.
  5. Get into BIOS by pressing F10 while booting.
    5.1 Disable security start up.
    5.2 Disable Virtualization Technology
    5.3 Disable CFG
  6. Start the installation with usb drive.

  7. You may need to format the hard drive.

  8. Then install the OS. There were several restarts during installation.

  9. After installation, restart the machine with USB and add option -disablegfxfirmware. That will prevent the grey screen during restart.
  10. Run the MultiBeast. Select UEFI and in the drive tap select the latest version of VoodooHDA. I got 2.9.0 in 06/24/18.
  11. Run EFI Mounter 3. Then get edit the clover /EFI/CLOVER/config.plist. Change the Intel video injection to false. (This is confusing. I found when this is on. OS can’t be started.)
  12. Install the RealtekRTL8100 driver with with KextBeast.
  13. Install the VoodooPS2Controller.kext with KextBeast and copy the VoodooPS2Daemon in the folder to /usr/bin
  14. Restart. All DONE!!!
